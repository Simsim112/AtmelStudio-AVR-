# AtmelStudio-AVR

## 1일차(GPIO)
- LED Module 점등
- 스위치로 LED 점등
- FND 출력확인
- 스위치로 FND 출력

## 2일차(GPIO + LCD)
- Interrupt를 사용하여 스위치로 LED 양쪽이동
- Inturrept를 사용하여 버튼으로 FND 초기화, 시작 
- LCD 모니터로 글자 출력
- LCD 모니터로 구구단 출력하면서 Interrupt 발생시 잠시 중지 

## 3일차(UART + Timer_COunter)
- UART를 사용해서 Hllo 보내기
- 스위치 입력 값 보내기
- PC 데이터 주고받기
- PC에서 FND점멸 
- 숫자 입력 받아서 구구단 출력
- 타이머를 사용해서 LED점멸
- 타이머를 이용한 디지털 시계
